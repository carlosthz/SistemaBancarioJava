# 🏦 Sistema Bancário em Java

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos (POO), encapsulamento, modificadores de acesso e organização de classes através da simulação de um sistema bancário via terminal.

---

## Funcionalidades

- Criação de contas bancárias
- Depósito
- Saque
- Alteração de titular
- Exibição de informações da conta
- Validação de entradas
- Controle de saldo com encapsulamento
- Estrutura orientada a objetos
